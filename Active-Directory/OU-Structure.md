# Organizational Unit Structure

## Objective

Create a logical Organizational Unit (OU) structure within Active Directory to support user administration, workstation management, server management, and departmental separation.

---

## Organizational Units Created

- IT
- HR
- Finance
- Workstations
- Servers

---

## Purpose

The OU structure provides administrative separation and enables future implementation of:

- User account management
- Security group administration
- Group Policy deployment
- Workstation management
- Server administration
- Department-specific access control

## Outcome

A structured Active Directory hierarchy was created to simulate a small enterprise environment. The design supports centralized administration and provides a foundation for user provisioning, computer management, and policy enforcement.
